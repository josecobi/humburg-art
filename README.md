# Artist Portfolio Website

A premium Next.js 14 artist portfolio website with modern design, responsive layout, and smooth animations.

## Features

- Next.js 14 with App Router, TypeScript, and Tailwind CSS
- Responsive mobile-first design with smooth scroll behavior
- Components: Header with minimal navigation, Hero section with video background, About section, Gallery with filtering, Contact section, Footer
- next/image for automatic optimization (WebP/AVIF conversion, lazy loading, responsive srcsets)
- Framer Motion for subtle scroll animations and page transitions
- Sophisticated color palette with neutrals and one elegant accent color
- Modern typography with serif headings and clean sans-serif body text
- Gallery with filtering capabilities
- Responsive design that works on all devices

## Dependencies

- framer-motion
- yet-another-react-lightbox

## Project Structure

```
/app
/components
/lib
/public
```

## How to Run

1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
3. Visit: http://localhost:3000

## Design Elements

- Color palette: sophisticated neutrals (off-white backgrounds, charcoal text, one elegant accent color)
- Typography: modern serif headings, clean sans-serif body text
- Responsive design with mobile-first approach
- Smooth animations using Framer Motion
- Optimized images using next/image
- Lightbox gallery with filtering capabilities
